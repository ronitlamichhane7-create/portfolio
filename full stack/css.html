Section Three

CSS creates the visual experience.

Colour, spacing, typography, layout, hover effects, transitions, and animations are controlled by CSS.

Explanation

CSS stands for Cascading Style Sheets. It is the language used to control the appearance, layout, spacing, colour, typography, animation, and responsive behaviour of a website. While HTML creates the structure of a page, CSS turns that structure into a designed user interface. CSS can change simple content into a polished and professional website by applying visual rules to elements. For example, CSS can make a navigation bar fixed at the top of the screen, create a modern card design, add shadows and rounded corners, control the size of headings, and adjust spacing between sections. It also allows developers to create layouts that respond to different screen sizes.

Good CSS improves both appearance and usability. A website should not only look attractive; it should also be clear, readable, and consistent. In this portfolio, CSS is used to create a dark modern design with gradient highlights, glass-style cards, animated backgrounds, hover effects, smooth transitions, and scroll reveal animations. The colour palette uses bright blue, purple, pink, and orange accents to create a professional technology style. CSS variables are used to make the design easier to maintain because colours and repeated values can be controlled from one place. Media queries are also used to make the website responsive on smaller devices. This demonstrates that CSS is not just decoration. It is an essential part of front-end development because it controls how users experience the content. A well-written stylesheet makes a website feel more polished, organised, memorable, and suitable for a professional portfolio.

Design Thinking

CSS is where a plain document becomes a polished visual product with mood, hierarchy, and personality.

Responsive CSS and variables

CSS variables make it simple to update the colour palette and spacing values across the whole website. Media queries allow the layout to adapt for mobile, tablet, and desktop screens so the design remains usable on all devices.

CSS Example Code

:root {
  --primary: #00d4ff;
  --background: #050816;
  --radius: 28px;
}

@media (max-width: 768px) {
  .content-grid {
    grid-template-columns: 1fr;
  }
}
